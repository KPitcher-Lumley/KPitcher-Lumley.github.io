ADIOS

Built by Joseph Limin, David Ma, and myself; ADIOS is a game case dispensing system built to make retreival of game cases easier. The project was built using our knowledge on embeded systems, circuit design and website design. The system consists of custom designed trays, hardware mounts, stepper and servo motors as well as other various other IC's all run by two ESP32's talking to each other over a serial conection.
